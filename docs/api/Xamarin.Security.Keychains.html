Namespace Xamarin.Security.Keychains

Classes

AppleKeychain

An IKeychain implementation backed by the macOS Keychain Services API for secure storage of secrets, scoped to the current user's login keychain.

DPAPIKeychain

An IKeychain implementation backed by the Windows Data Protection API for secure storage of secrets, scoped to the current user.

FileSystemKeychain

An abstract base class IKeychain implementation for keychains that store items as files in a directory structure, like DPAPIKeychain and ManagedProtectionKeychain.

IKeychainExtensions

ManagedProtectionKeychain

An IKeychain implementation backed by Mono's DPAPI-compatible Mono.Security.Cryptography.ManagedProtection.

OSKeychain

An IKeychain implementation backed by AppleKeychain on macOS, DPAPIKeychain on Windows, and ManagedProtectionKeychain on other platforms.
